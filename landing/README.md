# Landing Page

Generated by `idea-business/landing-page` pipeline (ADR-022).

## Dev

```bash
pnpm install
pnpm dev          # dev server @ localhost:3000
```

## Build (SSG)

```bash
pnpm build        # produces ./out/ — static HTML + JS bundles
```

`next.config.ts` 已设 `output: "export"`，`pnpm build` 产出纯静态 HTML，可直接扔任何 CDN / Vercel / Netlify 部署。

## 结构

```
src/
├── app/
│   ├── layout.tsx    — 全局布局（字体、Lenis smooth scroll）
│   ├── page.tsx      — 主页（由 gen-page skill 生成）
│   └── globals.css   — 设计 tokens + Tailwind
├── kit/              — 视觉 kit（由 gen-kit skill 生成）
│   ├── tokens.ts     — colors / typography / spacing
│   └── components/   — 基础 UI 原件
└── page/
    ├── providers.tsx — 客户端 provider（smooth scroll 等）
    └── hooks/        — 共享 hook
```

## 生成流程

每个此项目由 hub `landing-page/scripts/pipeline.ts` 跑 ADR-022 流程产生。
历史快照在 `projects/<slug>/.pipeline/landing/<runId>/snapshot/`。
